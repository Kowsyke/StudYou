Refined Design Tokens

A quiet, confident, native macOS visual foundation for international students.

StudYou uses a minimalist, content-first design system with generous spacing, soft depth, and restrained color. The canvas is a warm near-white background. Cards and panels use pure white with a soft border and subtle shadows. Colors are neutral grays, except for a single accent color used sparingly. No emojis are used in the user interface.

Motion Specification

Subtle, quick, and native transitions. Never bouncy, always confident.

Motion is used to reinforce spatial relationships, indicate interactivity, and celebrate completions without creating cognitive overhead or slowing down workflow.

Durations & Easings

Interaction Duration Easing Function Description
Hover states & focus rings 120ms linear Instant response to pointer location change.
Task checkbox tick & toast slide-in 200ms cubic-bezier(0.16, 1, 0.3, 1) Smooth, fast motion. The checkmark draws using a stroke offset.
Command palette open & transitions 300ms cubic-bezier(0.16, 1, 0.3, 1) A polished, swift macOS-like drop down/fade animation.
Dialog overlay fades 250ms cubic-bezier(0.65, 0, 0.35, 1) Calm, gradual appearance to prevent disorientation.
