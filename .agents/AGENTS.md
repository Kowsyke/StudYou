# Workspace Guidelines

## Dash Punctuation Rule (Iron Clad)
- **No Em Dashes (`—`)**: Never use em dashes (`—`) in text, documentation, comments, or code.
- **No Double Hyphens (`--`) as Punctuation**: Never use `--` as a dash or punctuation anywhere in text or code.
- Use single hyphens (`-`), colons (`:`), commas, or standard punctuation instead.
